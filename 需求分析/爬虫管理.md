#### 4.2.2 爬虫管理

| 功能名称     | 功能说明                                                           | 前置条件                                                                                        | 业务流程                                                                                                                                                                                                                     |                                                                                                                                                                                                                     | 异常处理 |
| -------- | -------------------------------------------------------------- | ------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---- |
|          |                                                                |                                                                                             | 用户动作                                                                                                                                                                                                                     | 系统响应                                                                                                                                                                                                                |      |
| 显示爬虫列表   | 爬虫管理的主页，展示用户的爬虫元数据信息，并提供编辑、查询、删除等选项                            | 1. 用户处于登录状态                                                                                 | 1. 点击“爬虫管理”按钮                                                                                                                                                                                                            | 1. 系统查询数据库，获取用户管理的爬虫信息，展示爬虫列表页面                                                                                                                                                                                     |      |
| 上传脚本创建爬虫 | 用户上传爬虫程序，填写执行方式等信息，使系统可以调用管理                                   | 1. 用户处于登录状态<br>    <br>2. 处于爬虫列表页面<br>    <br>3. 无同名爬虫                                      | 1. 点击"新增"按钮，选择"上传脚本"方式<br>    <br>2. 上传脚本文件<br>    <br>3. 填写爬虫名称、描述、执行方式、可配置参数等信息<br>    <br>4. 点击"确认创建"按钮                                                                                                               | 1. 系统展示创建爬虫表单<br>    <br>2. 系统接收脚本文件并存储<br>    <br>3. 统校验爬虫名称是否重复、文件格式是否合法<br>    <br>4. 校验通过后将爬虫信息写入数据库，返回创建成功，列表刷新显示新爬虫                                                                                           |      |
| 使用流程组件创建 | 用户使用现有的流程组件进行组合配置，无需手动编写脚本即可创建爬虫                               | 1. 用户处于登录状态<br>    <br>2. 处于爬虫列表页面<br>    <br>3. 无同名爬虫                                      | 1. 点击"新增"按钮，选择"流程组件"方式<br>    <br>2. 在可视化编辑器中拖拽组件、配置各组件参数、连接流程节点<br>    <br>3. 填写爬虫名称、描述等基本信息<br>    <br>4. 点击"保存"按钮                                                                                                     | 1. 系统展示可视化流程编辑器及组件库<br>    <br>2. 系统实时渲染流程图，支持组件参数的表单配置<br>    <br>3. 统校验爬虫名称是否重复<br>    <br>4. 校验流程是否完整（有起始节点、无断链） 通过后将爬虫配置序列化存储，返回创建成功，列表刷新显示新爬虫                                                                  |      |
| 配置流程     | 用户对已使用流程组件创建的爬虫进行编辑，修改组件配置或流程结构                                | 1. 用户处于登录状态<br>    <br>2. 处于爬虫列表页面<br>    <br>3. 目标爬虫存在且为流程组件类型<br>    <br>4. 该爬虫当前无正在执行的任务 | 1. 点击目标爬虫的"配置流程"按钮<br>    <br>2. 在可视化编辑器中修改组件配置或流程结构<br>    <br>3. 点击"保存"按钮                                                                                                                                              | 1. 系统加载当前爬虫的流程配置，渲染至可视化编辑器<br>    <br>2. 系统实时渲染流程图，支持组件参数的表单配置<br>    <br>3. 校验修改后的流程完整性 校验通过后更新数据库中的爬虫配置，返回保存成功                                                                                                    |      |
| 查询爬虫信息   | 用户通过输入爬虫名称、状态、创建时间等元数据条件，筛选查询符合条件的爬虫列表                         | 1. 用户处于登录状态<br>    <br>2. 处于爬虫列表页面                                                          | 1. 在爬虫管理页面的搜索区域输入查询条件（名称、状态、时间范围等）<br>    <br>2. 点击"搜索"按钮                                                                                                                                                                | 2. 系统根据查询条件检索数据库 返回符合条件的爬虫列表，无结果时展示空状态提示                                                                                                                                                                            |      |
| 编辑爬虫     | 用户配置爬虫参数，例如Header，请求方法，请求URL等。用户还可以设置爬虫的执行方式支持手动触发、定时任务（Cron）等 | 1. 用户处于登录状态<br>    <br>2. 处于爬虫列表页面<br>    <br>3. 目标爬虫存在<br>    <br>4. 该爬虫当前无正在执行的任务         | 1. 点击目标爬虫的"配置流程"按钮<br>    <br>2. 在参数表单中修改对应字段的值<br>    <br>3. 点击"保存"按钮<br>    <br><br>---<br><br>1. 点击目标爬虫的"配置流程"按钮<br>    <br>2. 在"执行方式"区域选择执行类型（手动 / 定时）<br>    <br>3. 若选择定时，填写 Cron 表达式或选择时间周期<br>    <br>4. 点击"保存"按钮 | 1. 系统加载当前爬虫的参数配置并展示在表单中<br>    <br>2. 系统校验参数格式合法性<br>    <br>3. 校验通过后更新数据库中的参数配置，返回保存成功<br>    <br><br>---<br><br>1. 系统展示执行方式配置表单<br>    <br>2. 系统校验 Cron 表达式格式是否合法<br>    <br>3. 校验通过后更新爬虫执行配置，若为定时任务则注册调度器，返回保存成功 |      |
| 删除爬虫     | 用户删除指定爬虫及其相关配置，释放系统资源                                          | 1. 用户处于登录状态<br>    <br>2. 处于爬虫列表页面<br>    <br>3. 目标爬虫存在<br>    <br>4. 该爬虫当前无正在执行的任务         | 1. 点击目标爬虫的"删除"按钮<br>    <br>2. 在弹出的确认对话框中点击"确认删除"                                                                                                                                                                        | 1. 系统弹出确认对话框，提示删除操作不可撤销<br>    <br>2. 用户确认后，系统删除该爬虫的配置信息及关联调度任务，返回删除成功，列表刷新                                                                                                                                         |      |
| 手动执行爬虫   | 用户主动触发指定爬虫立即运行，生成一条执行任务记录                                      | 1. 用户处于登录状态<br>    <br>2. 处于爬虫列表页面<br>    <br>3. 目标爬虫存在且配置完整<br>    <br>4. 该爬虫当前无正在执行的任务    | 1. 点击目标爬虫的"立即执行"按钮<br>    <br>2. 在弹出的确认对话框中点击"确认执行"                                                                                                                                                                      | 1. 系统弹出确认对话框<br>    <br>2. 用户确认后，系统创建一条执行任务记录（状态为"执行中"），调用调度器触发爬虫运行，返回执行已启动的提示，并可跳转至任务记录页查看进度                                                                                                                       |      |
