#### 4.2.4 舆情分析

| 功能名称     | 功能说明                                                  | 前置条件                                                       | 业务流程                                             |                                                                                                   | 异常处理 |
| -------- | ----------------------------------------------------- | ---------------------------------------------------------- | ------------------------------------------------ | ------------------------------------------------------------------------------------------------- | ---- |
|          |                                                       |                                                            | 用户动作                                             | 系统响应                                                                                              |      |
| 显示舆情报告   | 显示用户创建的舆情报告，提供包括名称、类型、监测关键词、生成时间以及相关操作                | 1. 用户已登录                                                   | 1. 点击“舆情分析”按钮                                    | 1. 系统查询数据库，获取用户管理的舆情报告，展示舆情报告页面                                                                   |      |
| 查询舆情报告   | 通过报告名称、关键词和报告类型查询想要查看的舆情报告                            | 1. 用户已登录<br>    <br>2. 处于舆情分析页面                            | 1. 在搜索区输入报告名称、关键词、报告类型<br>    <br>2. 点击"查询"按钮    | 1. 根据条件查询舆情报告表，返回符合条件列表，无结果展示空状态                                                                  |      |
| 查看舆情报告   | 显示舆情报告具体信息、展示分析周期、生成时间以及报告具体内容，对具体内容提供编辑与删除功能         | 1. 用户已登录<br>    <br>2. 处于舆情分析页面                            | 1. 点击相关舆情报告的“查看”按钮                               | 1. 系统查询舆情报告表返回报告详情到前端                                                                             |      |
| 编辑舆情报告   | 在展示已有的舆情报告相关信息下，提供对不同信息类型下的相关内容进行编辑                   | 1. 用户已登录<br>    <br>2. 处于舆情分析页面<br>    <br>3. 处于舆情报告详细信息页面 | 1. 点击舆情报告不同信息类型的“编辑”按钮                           | 1. 加载当前报告信息<br>    <br>2. 前端将包装为接口数据向java后端发送请求，java后端将预警专题信息存入数据库<br>    <br>3. 返回前端“舆情报告内容修改成功” |      |
| 删除舆情报告   | 删除已创建的舆情报告                                            | 1. 用户已登录<br>    <br>2. 处于舆情分析页面<br>    <br>3. 存在相关舆情报告记录   | 1. 点击相关舆情报告的“删除”按钮<br>    <br>2. 点击"确认删除"按钮      | 1. 系统弹出确认对话框，提示删除操作将同时清除关联日志和数据结果且不可撤销<br>    <br>2. 用户确认后，Java后端删除舆情报告表中该舆情报告及其关联数据，返回删除成功，列表刷新  |      |
| 显示舆情报告专题 | 显示用户创建的报告专题，用户可以查看报告专题的监测内容、监测时段以及创建时间，用户可以对专题进行编辑和删除 | 1. 用户已登录<br>    <br>2. 处于舆情分析页面<br>    <br>3. 存在相关舆情报告记录   | 1. 点击舆情分析中“舆情报告设置”                               | 1. 系统查询数据库，获取用户管理的舆情报告专题，展示舆情报告专题列表页面                                                             |      |
| 新增舆情报告   | 新增舆情报告，填写报告名称，监测词组、选择报告类型和监测时间等相关信息                   | 1. 用户已登录<br>    <br>2. 处于舆情分析页面                            | 1. 点击舆情分析中“舆情报告设置”<br>    <br>2. 点击”新增舆情报告“      | 1. 系统弹出新增信息提示框，用户填写完毕信息后，将报告相关信息写入数据库<br>    <br>2. 在舆情报告的显示中新增此报告                                |      |
| 编辑舆情报告专题 | 显示用户创建的原有报告专题相关信息，用户可以对其中信息进行编辑                       | 1. 用户已登录<br>    <br>2. 处于舆情分析页面<br>    <br>3. 存在相关舆情报告专题   | 1. 点击舆情分析中“舆情报告设置”<br>    <br>2. 点击相关舆情报告的”编辑按钮“ | 1. 系统查询数据库，获取用户原有的舆情报告专题，展示舆情报告专题列表页面<br>    <br>2. 系统获取修改后的相关信息，重新写入数据库                          |      |
