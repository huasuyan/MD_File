#### 4.2.6 数据大屏

| 功能名称       | 功能说明                                             | 前置条件     | 业务流程        |                                    | 异常处理 |
| ---------- | ------------------------------------------------ | -------- | ----------- | ---------------------------------- | ---- |
|            |                                                  |          | 用户动作        | 系统响应                               |      |
| 显示信息总量统计数据 | 用户进入数据大屏后，可以查看到信息总量、预警信息总量、办件总量、今日信息量、今日预警信息量等数据 | 1. 用户已登录 | 1. 进入数据大屏页面 | 1. 调用后端接口，获取统计数据，返回前端页面展示，可以展示趋势数据 |      |
| 地域分布情况展示   | 用户进入数据大屏后，可以看到预警信息/舆情报告信息的地域分布情况                 | 1. 用户已登录 | 1. 进入数据大屏页面 | 1. 调用后端接口，获取地域分布情况数据，返回前端页面展示      |      |
| 热词分析       | 用户进入数据大屏后，可以看到预警消息的热词分析                          | 1. 用户已登录 | 1. 进入数据大屏页面 | 1. 调用后端接口，获取热词分析数据，返回前端页面展示        |      |
| 平台分布情况展示   | 用户进入数据大屏后，可以看到舆情信息的发布平台发布情况                      | 1. 用户已登录 | 1. 进入数据大屏页面 | 1. 调用后端接口，获取平台分布情况数据，返回前端页面展示      |      |
