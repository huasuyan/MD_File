|           |                                          |                                                                     |     | 用户动作                                                                         | 系统响应                                                                                            |      |
| 显示任务记录列表  | 显示出当前用户所有的任务记录                           | 1. 用户处于登录状态                                                         |     | 1. 点击“任务记录”按钮                                                                | 1. 系统查询数据库，获取用户爬虫的任务记录信息，展示任务记录列表页面                                                             |      |
| 查询任务记录    | 用户通过输入爬虫名称、执行结果、爬虫类型等元数据条件，筛选查询符合条件的爬虫列表 | 1. 用户处于登录状态                                                         |     | 1. 在任务记录列表页面的搜索区域输入查询条件（爬虫名称、状态、时间范围等）<br>    <br>2. 点击"搜索"按钮                | 2. 系统根据查询条件检索数据库 返回符合条件的爬虫列表，无结果时展示空状态提示                                                        |      |
| 查看或导出执行日志 | 查看指定任务的执行日志，可以选择导出该执行日志                  | 1. 用户处于登录状态<br>    <br>2. 处于任务记录列表界面<br>    <br>3. 存在执行完成的任务        |     | 1. 点击任务的“操作”栏中的“执行日志”按钮<br>    <br>2. 点击“导出”按钮                               | 2. 系统读取该任务对应的执行日志 显示该执行日志信息<br>    <br>3. 系统将日志信息导出为txt文件                                       |      |
| 操作数据结果    | 用户对任务采集到的数据结果进行查看、导出等操作                  | 1. 用户处于登录状态<br>    <br>2. 处于任务记录列表界面<br>    <br>3. 存在执行完成的并且执行成功的任务 |     | 1. 点击执行成功的任务的“数据结果”按键<br>    <br>2. 点击“导出”按钮，选择导出格式<br>    <br>3. 点击"确认导出"按钮 | 1. 系统查询数据库并返回数据结果 系统展示该任务采集数据的预览列表<br>    <br>2. 系统按所选格式打包数据文件<br>    <br>3. 系统返回下载链接，浏览器触发文件下载 |      |
| 删除任务记录    | 删除任务记录和执行结果数据                            | 1. 用户处于登录状态<br>    <br>2. 处于任务记录列表界面<br>    <br>3. 目标任务存在且状态不为"执行中" |     | 1. 点击任务的的“删除”按钮<br>    <br>2. 在弹出的确认对话框中点击"确认删除"                             | 1. 系统弹出确认对话框，提示删除操作将同时清除关联日志和数据结果且不可撤销<br>    <br>2. 用户确认后，系统删除数据库中该任务记录及其关联数据，返回删除成功，列表刷新      |      |
