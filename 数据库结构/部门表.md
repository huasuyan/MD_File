## 部门表

dept

|字段|数据类型|约束|说明|
|---|---|---|---|
|dept_id|BIGINT|NOT NULL AUTO_INCREMENT|部门ID|
|parent_dept_id|BIGINT|DEFAULT 0|上级部门ID 0=顶级|
|dept_name|VARCHAR(100)|NOT NULL|部门/局室名称|
|dept_level|int|NOT NULL|层级 0:系统管理 1:市局 2:部门 3:处室|
|status|int|NOT NULL DEFAULT 1|状态 0:停用 1:启用|
|create_time|DATETIME|NOT NULL DEFAULT CURRENT_TIMESTAMP||
|update_time|DATETIME|NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP||
