## 角色权限表

role

|字段|数据类型|约束|说明|
|---|---|---|---|
|role_id|BIGINT|NOT NULL AUTO_INCREMENT|角色ID|
|dept_id|BIGINT|NOT NULL|绑定的部门ID|
|role_name|VARCHAR(50)|NOT NULL|角色名称|
|data_scope|INT|NOT NULL|数据权限范围 1:本处室 2:本单位 3:本系统|
|remark|VARCHAR(255)|""|备注说明|
|status|int|NOT NULL<br><br>DEFAULT 1|状态 0:禁用 1:启用|
|authority|json|NOT NULL|角色权限<br><br>{<br><br>"alert":{<br><br>  "alert_insert":1<br><br>  "alert_delete":1<br><br>  "alert_update":1<br><br>  "alert_select":1<br><br>  },<br><br>"role":{<br><br>  "role_select":0<br><br>  "role_create":0<br><br>  "role_update":0<br><br>  "role_delete":0<br><br>  },<br><br>"XX":{<br><br>  ......<br><br>}<br><br>}<br><br>0表示无权限，1表示有权限<br><br>alert_insert：舆情预警模块增加权限，<br><br>alert_delete：舆情预警模块删除权限，<br><br>alert_update：舆情预警模块修改权限，<br><br>alert_select：舆情预警模块查询权限，<br><br>role_select：角色模块查询权限，<br><br>role_create：角色模块新增权限，<br><br>role_update：角色模块编辑权限，<br><br>role_delete：角色模块删除权限|
|create_time|DATETIME|NOT NULL DEFAULT CURRENT_TIMESTAMP||
|update_time|DATETIME|NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP||
