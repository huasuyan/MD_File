## 原始爬取数据表

news_data

| 字段                | 数据类型          | 约束                                | 说明            |
| ----------------- | ------------- | --------------------------------- | ------------- |
| news_id           | bigint        | 非空、自增                             | 数据唯一ID        |
| special_report_id | bigint        | 默认为0                              | 数据对应的舆情报告设置ID |
| is_report_need    | int           | 默认0                               | 是否是需要生成报告的新闻  |
| alert_id          | bigint        | 默认为0                              | 数据对应的预警专题ID   |
| title             | varchar(500)  | 非空                                | 标题            |
| content           | longtext      | 默认为null                           | 内容            |
| video             | varchar(1000) | 默认null                            | 视频链接          |
| platform          | varchar(100)  | 默认null                            | 平台            |
| source            | varchar(100)  | 默认null                            | 来源            |
| publisher         | varchar(100)  | 默认null                            | 发布人           |
| publish_time      | DATETIME      | 非空                                | 发布时间          |
| comment           | int           | 默认0                               | 评论数           |
| region            | varchar(100)  | 默认null                            | 地域            |
| original_url      | varchar(1000) | 非空                                | 原文地址          |
| article_type      | varchar(100)  | 默认null                            | 发文类型          |
| source_url        | varchar(100)  | 默认null                            | 来源地址          |
| create_time       | datetime      | DEFAULT CURRENT_TIMESTAMP COMMENT | 入库时间          |
