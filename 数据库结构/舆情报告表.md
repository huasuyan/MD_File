## 舆情报告表

report_result

| 字段                       | 数据类型         | 约束                                   | 说明       |
| ------------------------ | ------------ | ------------------------------------ | -------- |
| report_id                | BIGINT       | NOT NULL、主键、自增                       | 舆情报告ID   |
| special_report_id        | BIGINT       | NOT NULL、外键关联special_report_setting表 | 舆情报告专题ID |
| report_name              | VARCHAR(255) | NOT NULL                             | 报告名称     |
| monitor_keywords         | JSON         | NOT NULL                             | 监测词组     |
| report_type              | VARCHAR(50)  | NOT NULL                             | 报告类型     |
| brief_summary            | TEXT         | 非必填                                  | 简报概述     |
| monitor_summary          | TEXT         | 非必填                                  | 监测概述     |
| opinion_trend            | TEXT         | 非必填                                  | 舆情发展趋势   |
| source_media_analysis    | TEXT         | 非必填                                  | 来源媒体分析   |
| emotion_analysis         | TEXT         | 非必填                                  | 情绪分析     |
| region_distribution      | TEXT         | 非必填                                  | 地域分布     |
| region_distribution_list | JSON         | 非空                                   | 地域分布数据集合 |
| hot_analysis_words       | TEXT         | 非必填                                  | 热分析词     |
| hot_analysis_words_list  | JSON         | 非必填                                  | 热分析词数据集合 |
| hot_information          | TEXT         | 非必填                                  | 热门信息     |
| disposal_opinions        | TEXT         | 非必填                                  | 处置意见     |
| create_time              | DATETIME     | 非空、默认当前时间                            | 创建时间     |
| update_time              | DATETIME     | 非空、更新时自动刷新                           | 更新时间     |
