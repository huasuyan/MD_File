## 办件处理表

| 字段            | 数据类型         | 约束                 | 说明                                                     |
| ------------- | ------------ | ------------------ | ------------------------------------------------------ |
| case_id       | bigint       | NOT NULL<br><br>主键 | 办件唯一ID                                                 |
| user_id       | bigint       | NOT NULL           | 外键，用户ID                                                |
| dept_id       | bigint       | NOT NULL           | 外键，部门ID，在创建时由用户设置                                      |
| case_name     | varchar(50)  | NOT NULL           | 办件名称                                                   |
| case_info     | varchar(300) |                    | 办件描述                                                   |
| case_level    | int          | NOT NULL           | 办件登记<br><br>0：紧急<br><br>1：高<br><br>2：中<br><br>3：一般     |
| money         | decimal      |                    | 涉案金额                                                   |
| news_id       | bigint       | NOT NULL           | 对应的预警记录ID                                              |
| trigger_state | int          | NOT NULL           | 办件是否关闭<br><br>0：已关闭<br><br>1：已启用                       |
| state         | int          | NOT NULL           | 办件状态<br><br>0：待办<br><br>1：办理中<br><br>2：已归档<br><br>3：异常 |
| create_time   | DATETIME     |                    | 创建时间，默认当前时间                                            |
| update_time   | DATETIME     |                    | 上次修改时间                                                 |
