## 数据清洗表

clear_data

| 字段                | 数据类型        | 约束            | 说明                                                                   |
| ----------------- | ----------- | ------------- | -------------------------------------------------------------------- |
| clear_id          | bingint     | 主键            | 清洗数据ID                                                               |
| news_id           | bigint      | 外键（news_data） | 对应新闻ID                                                               |
| sensitivity_level | int         | 默认 0          | 敏感度等级：<br><br>0 = 普通、<br><br>1 = 低敏感、<br><br>2 = 中敏感、<br><br>3 = 高敏感 |
| sensitivity_label | varchar(50) | 可为空           | 敏感标签分类（政治、民生、金融、负面、谣言等）                                              |
| sentiment_type    | int         | 默认 0          | 情绪极性：-1 = 负面、0 = 中性、1 = 正面                                           |
