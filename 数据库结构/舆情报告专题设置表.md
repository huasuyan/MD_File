## 舆情报告专题设置表

special_report_setting

|字段|数据类型|约束|说明|
|---|---|---|---|
|special_report_id|bigint|主键、自增、NOT NULL|舆情报告专题ID|
|report_name|varchar(200)|NOT NULL|报告专题名称|
|create_user_id|int|NOT NULL、外键关联User表|创建人ID|
|monitor_keywords|json|默认{}|监测词组|
|data_source|varchar(500)|可为空|内容来源平台（网易、中华网等）|
|params|json|默认{}|可变参数|
|monitor_region|varchar(300)|可为空|监测地域|
|report_type|int|非空|报告类型：<br><br>1：即时报告 2：定时报告|
|type_params|json|默认{}|类型参数<br><br>_即时报告(reportType=1)：_ _{"start_date":"2026-04-20","end_date":"2026-04-30"}_<br><br>_定时报告(reportType=2) 按天：_ _{"cycle":"daily","time":"08:00"}_<br><br>  <br><br>_定时报告(reportType=2) 按周：_ _{"cycle":"weekly","weekday":1,"time":"08:00"}_<br><br>  <br><br>_定时报告(reportType=2) 按月：_ _{"cycle":"monthly","day":7,"time":"08:00"}_|
|status_enabled|int|NOT NULL 默认: 0|启用状态：<br><br>1 = 启用，<br><br>0 = 停用|
|create_time|datetime|NOT NULL DEFAULT CURRENT_TIMESTAMP|创建时间|
|execute_status|int|NOT NULL DEFAULT 0|执行状态|
|last_execute_time|datetime|可为空|上一次报告生成执行时间|
|last_upate_time|datetime|可为空|上一次爬虫更新时间|
